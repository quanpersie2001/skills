# Dev Note Topics Index Template

Use this template when creating or rebuilding `dev-notes/distil/TOPICS.md`.

**Save to:** `dev-notes/distil/TOPICS.md`

Rules for filling this in:
- This is the global TOC/index for all distilled dev-note topics.
- Rebuild it from the current topic files instead of editing it ad hoc.
- Keep each topic entry to one line plus one short hook.
- Keep links repository-relative.
- Sort topics alphabetically by slug unless the workflow explicitly says otherwise.
- Do not add prose sections outside this template.

---

```markdown
# Dev Note Topics

This index lists all distilled developer-learning topics.

- `<topic-slug>` → `topics/<topic-slug>/<topic-slug>.md` — <one-line description>. Notes: <integer>
```
